# Flight-management-system
本项目是一个基于Spring Boot和Vue.js的航班预订系统，采用前后端分离架构设计。根据角色的权限不同，系统为普通用户提供了航班查询、乘客管理和订单查询等功能，支持普通用户查询航班、预订机票、管理个人信息等操作。同时为管理员提供了航班管理、用户管理订单管理以及数据统计等功能，满足了管理员对后台数据的管理需求。
